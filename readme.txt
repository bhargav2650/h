this is a blog app implemented using nodejs, react and mongodb

the routes are
1) to create posts: /POST/posts
2) to view all posts: /GET/posts         (output in console of browser)
3) to view only one pose: /GET/posts/:id   (output in console of browser)
4) to modify any post:/PUT/posts/:id
5) to delete any post:/DELETE/posts/:id

also jwt implementation done too.

run client using "yarn start"
run server(app) using "nodemon index"
